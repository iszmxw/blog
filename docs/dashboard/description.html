# 介绍

博客的后台采用PHP框架中的laravel5.8以及vue-admin-admin搭建 前台采用一款比较清新的主题，该主题是模仿的handsome主题

# 环境

  • linux+nginx+mysql+php
  • nginx/1.14.1
  • PHP7.2+
  • MySql5.6+
